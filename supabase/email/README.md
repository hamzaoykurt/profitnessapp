# Profitness Auth Email Templates

Profitness auth emails are managed through Supabase Auth templates. The source templates in
`templates/` are generated by `apply-auth-email-templates.ps1` and mirror the live hosted
Supabase project configuration.

The template set covers:

- Confirm signup (`confirmation.html`) with the 6-digit OTP code
- Reset password (`recovery.html`) with `{{ .ConfirmationURL }}`
- Magic link (`magic_link.html`) with `{{ .ConfirmationURL }}`
- Invite (`invite.html`) with `{{ .ConfirmationURL }}`
- Email change (`email_change.html`) with `{{ .ConfirmationURL }}`
- Reauthentication (`reauthentication.html`) with the OTP code

The logo is served from the public `email-assets` Edge Function:

```text
https://dkcriptafzdrynsilxku.supabase.co/functions/v1/email-assets/logo.png
```

To regenerate files without touching the hosted project:

```powershell
.\supabase\email\apply-auth-email-templates.ps1 -WriteOnly
```

To apply the templates to hosted Supabase, set `SUPABASE_ACCESS_TOKEN` first:

```powershell
$env:SUPABASE_ACCESS_TOKEN = "<token>"
.\supabase\email\apply-auth-email-templates.ps1
```
